# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-04

### Added

- Initial release
- `TalkingHead` component with real-time lip-sync
- `TalkingHeadWithTTS` all-in-one component
- `useTTS` hook for TTS synthesis
- `deriveAvatarState` helper for state management
- Vite plugin (`talkyHeadsPlugin`) for zero-config setup
- Bundled "Lucy" avatar (~6MB)
- Support for ElevenLabs and Google Cloud TTS
- Proxy mode for secure API key handling
- Circle and square shape options
- Automatic speaking state management
